== Description ==
Track WooCommerce orders using Paystack reference and verify payments for cancelled or pending Paystack orders.

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/verify-payments-for-orders-on-paystack/`
2. Activate the plugin.
3. For multisite, network activate it.
